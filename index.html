LLM Use Case

The Large Language Model part of our application uses LLMs to perform advanced analysis of specific user set data. Under the hood we are using the Google Gemini API. Because we use LLMs to perform the analysis, we are able to provide our findings in automatically generated human understandable text.

Data Analysis and Visualization

The Data Analysis and Visualization part of our application is designed to provide the user an easy and interactive way to visualize the data using multiple different types of charts and graphs. We hope this will create an ease of use so that the users can focus on using the insights for their benefit.
